CSS注释

CSS注释

和大多数语言一样，CSS规则是使用/* 需要注释的内容 */进行注释的，即在需要注释的内容前使用 /* 标记开始注释，在内容的结尾使用 */结束。

例如：

p {
    font-size: 14px; /* 所有的字体是14像素大小*/
}
